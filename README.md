# 🏃 오늘의 운동 기록 (Daily Exercise Tracker)

## 📖 앱 소개

**오늘의 운동 기록**은 운동량을 쉽고 빠르게 기록할 수 있는 Streamlit 기반의 웹 앱입니다.

### 주요 기능

- ✅ **빠른 입력**: 운동 종류, 시간, 거리, 걸음 수를 한 번에 입력
- ✅ **자동 칼로리 계산**: 운동 유형과 강도에 따른 예상 칼로리 자동 계산
- ✅ **직관적인 UI**: 슬라이더, 라디오 버튼, 입력창으로 쉬운 조작
- ✅ **운동 요약**: 오늘의 운동 정보를 한눈에 확인
- ✅ **동기 부여**: 운동 기록 후 칭찬 메시지와 격려문구 제공
- ✅ **데이터 저장**: 매일의 운동 기록을 JSON 파일로 자동 저장

## 🎯 사용 시나리오

- 학생: 체육시간과 방과후 운동을 빠르게 기록
- 직장인: 출근 전, 퇴근 후 운동량을 간편하게 입력
- 운동 애호가: 매일 저녁 하루의 운동량을 정리하고 확인

## 🛠️ 설치 방법

### 필수 요구사항
- Python 3.8 이상
- pip (Python 패키지 관리자)

### 1단계: 저장소 클론 또는 파일 다운로드

```bash
# 현재 디렉토리로 이동
cd /workspaces/classstreamit
```

### 2단계: 가상환경 생성 (선택사항이지만 권장)

```bash
# macOS / Linux
python3 -m venv .venv
source .venv/bin/activate

# Windows
python -m venv .venv
.venv\Scripts\activate
```

### 3단계: 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

## 🚀 실행 방법

### 방법 1: Streamlit 커맨드로 실행 (권장)

```bash
streamlit run main.py
```

실행 후 자동으로 브라우저가 열리며, 보통 `http://localhost:8501`에서 접속할 수 있습니다.

### 방법 2: 커스텀 포트로 실행

```bash
streamlit run main.py --server.port 8502
```

### 방법 3: 원격 서버에서 실행

```bash
streamlit run main.py --server.address 0.0.0.0
```

## 📊 앱 사용 가이드

### 1. 운동 정보 입력

| 필드 | 설명 | 예시 |
|------|------|------|
| **운동 종류** | 운동의 이름을 자유롭게 입력 | 걷기, 달리기, 요가 |
| **운동 시간** | 운동한 시간(분) | 30, 60, 90 |
| **걸음 수** | 운동 중 걸은 걸음 수 | 5000, 10000 |
| **거리** | 운동한 거리(km) | 2.5, 5.0 |
| **운동 강도** | 약함, 보통, 강함 중 선택 | 보통 |
| **운동 유형** | 미리 정의된 유형 중 선택 | 걷기, 달리기, 사이클 등 |

### 2. 버튼 기능

- **💾 기록 저장**: 입력한 운동 정보를 저장하고 칼로리 계산
- **🔄 초기화**: 오늘의 모든 기록 삭제
- **📊 결과 보기**: 저장된 운동 정보 조회 및 요약 확인

### 3. 결과 확인

기록 저장 후 또는 결과 보기를 클릭하면:
- 운동 종류, 시간, 칼로리, 걸음 수 표시
- 예상 칼로리 소모량 표시
- 운동에 따른 동기 부여 메시지
- 기록 저장 시간 표시

## 🧮 칼로리 계산 방식

앱은 운동 종류와 강도에 따라 자동으로 칼로리를 계산합니다:

| 운동 유형 | 분당 기본 칼로리 |
|----------|-----------------|
| 걷기 | 4 kcal |
| 달리기 | 10 kcal |
| 사이클 | 8 kcal |
| 수영 | 11 kcal |
| 스트레칭 | 3 kcal |

강도에 따른 배수:
- 약함: ×0.8
- 보통: ×1.0 (기본값)
- 강함: ×1.3

**계산 공식**: 운동 시간(분) × 기본 칼로리 × 강도 배수 = 예상 칼로리

## 📁 데이터 저장

- 모든 운동 기록은 `exercise_data/` 디렉토리에 저장됩니다
- 파일명: `YYYY-MM-DD.json` (예: 2026-05-21.json)
- JSON 형식으로 저장되어 다른 프로그램에서도 열 수 있습니다

### 저장된 데이터 예시

```json
{
  "timestamp": "2026-05-21 14:30:45",
  "exercise_name": "조깅",
  "duration": 30,
  "steps": 5000,
  "distance": 2.5,
  "intensity": "보통",
  "exercise_type": "달리기",
  "calories": 300
}
```

## 🎨 기능 특징

### 1. 반응형 디자인
- 모바일, 태블릿, 데스크톱 모두 최적화
- 쉬운 터치 입력

### 2. 실시간 입력 피드백
- 슬라이더로 직관적인 강도 선택
- 라디오 버튼으로 운동 유형 빠른 선택

### 3. 자동 동기 부여
- 30분 이상: "30분 운동 완료! 꾸준히 가고 있어요!"
- 60분 이상: "오늘 1시간 이상 운동했어요! 정말 멋져요!"
- 500kcal 이상: "500kcal 이상 소모했어요! 우와!"

### 4. 데이터 지속성
- JSON 파일로 자동 저장
- 앱 재실행 후에도 데이터 유지
- 초기화 버튼으로 언제든 삭제 가능
